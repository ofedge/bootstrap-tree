调用: $('#tree-demo').bootstrapTree();

参数
  • openOnLoad: 默认false, 初始化时候是否展开树
  • icons: 图标
    • openedNodeIcon: 节点打开时候的图标, 默认 glyphicon-minus
    • closedNodeIcon: 节点关闭时候的图标, 默认 glyphicon-plus
    • leafNodeIcon: 叶子节点图标, 默认 glyphicon-leaf

使用

$('#tree').bootstrapTree({
    openOnLoad: false,
    icons: {
        openedNodeIcon: 'glyphicon-folder-open',
        closedNodeIcon: 'glyphicon-folder-close',
        leafNodeIcon: 'glyphicon-file'
    }
});
方法
  • openAll: 打开所有节点
  • closeAll: 关闭所有节点

使用

$('#tree').bootstrapTree('openAll');
$('#tree').bootstrapTree('closeAll');
